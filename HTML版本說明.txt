🌐 HTML 版本使用說明
====================

✅ 優點：
- 不需要安裝 Python
- 不需要安裝任何套件
- 直接在瀏覽器中運行
- 資料不會上傳到任何伺服器（完全本地運行）

🚀 使用方式
==========

方法一：雙擊打開
---------------
直接雙擊「suno-subtitle-downloader.html」檔案
瀏覽器會自動開啟工具

方法二：拖曳到瀏覽器
------------------
將 HTML 檔案拖曳到瀏覽器視窗中

方法三：右鍵選擇瀏覽器
--------------------
右鍵點擊 HTML 檔案 > 選擇「打開方式」> 選擇瀏覽器

📝 使用步驟
==========
1. 在瀏覽器中登入 https://suno.com
2. 按 F12 開啟開發者工具
3. 找到 Cookies > https://suno.com > __session
4. 複製 __session 的值
5. 在 HTML 工具中：
   - 貼上 Suno 歌曲 URL
   - 貼上 Session Cookie
   - 輸入輸出檔名（選填，留空則使用歌曲 ID）
   - 點擊「開始下載」

✨ 功能特點
==========
- 🎨 美觀的現代化界面
- 📊 即時進度顯示
- 📝 執行日誌記錄
- ✅ 成功/錯誤提示
- 📱 響應式設計（支援手機）

💡 提示
======
- 建議使用 Chrome、Edge 或 Firefox
- 如果下載被阻擋，請允許瀏覽器下載檔案
- 所有處理都在你的瀏覽器中完成，不會上傳任何資料

🎉 就是這麼簡單！
現在你可以直接使用 HTML 版本，不需要安裝任何東西！
